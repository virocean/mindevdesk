# diweb+mindevdesk

mindevdesk os是一个基于debian，集成了devops和日用APP的系统，是一个云虚拟机系统加云chromeos/webos系统打造的类似业界多场景OS，
diweb.sh是mindevdesk os的recovery，及一个生成（ci.sh）和安装mindevdesk os镜像的脚本，类似群晖的webassiant
